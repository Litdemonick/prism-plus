Extensiones

Creá, probá y administrá las extensiones del catálogo de Prism+.

Publicar

Compilá (firma con tu llave) y subí los cambios a GitHub con un clic.

Compilar + firmar

Ejecuta npm run build: valida, compila a dist/, firma cada bundle con tu llave privada y regenera index.json. Hacé esto antes de probar o publicar.

Subir a GitHub

Hace git add/commit/push a main con tu autoría. Los usuarios de PrismHub reciben las extensiones nuevas/actualizadas vía index.json.

Salida

Seguridad / Llaves

La firma garantiza que nadie pueda falsificar una extensión oficial de Prism+.

Estado de la firma

PrismHub trae embebida tu llave pública y rechaza cualquier extensión que no esté firmada con tu llave privada. La privada vive solo en tu máquina (.keys/, gitignored) — respaldala.

¿Cómo protege esto a PrismHub?

• Oficial (firma válida) → se instala como verificada ✅
• Alterada (firma inválida) → PrismHub la rechaza ❌
• Externa (sin firma) → permitida pero marcada como no oficial ⚠️ (PrismHub es open source)
• Un sideload que duplica una extensión oficial → bloqueado
